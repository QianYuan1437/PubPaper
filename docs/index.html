CBF Focus

CBF / 安全控制方向推荐去向

如果你的工作核心围绕 Control Barrier Function、安全约束、可达性、安全滤波或安全强化学习，可优先考虑以下组合。

理论导向

TAC、L-CSS、CDC 适合偏理论证明、稳定性分析、可行性条件与泛化型 CBF 结果。

工程导向

TCST、CCTA、ACC 更适合带有机器人、车辆、无人机、工业系统实验验证的工作。

中文传播

CCC、CAC、CPCC 适合先做国内交流、积累反馈，再面向国际期刊或国际会议扩展投稿。
